Welcome to Order Flow System

This sample demonstrates how to use Spring MVC and Spring Web Flow together with Thymeleaf and Tiles.

Modules include:

  • Order using Spring Web Flow
  • Security and User Registration
  • JPA Reporitory and Data Service
  • Thymeleaf and HTML as Front end Solution
  • Javascript, AJAX integrated
  • SpringSource Tool Suite integration, with support for graphical flow modeling and visualization
